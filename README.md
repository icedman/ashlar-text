This app is a simple SublimeText replacement.
It uses the textmate parser. And reuses some visual studio code extensions.

# features
* syntax highlighting
* keybindings
* scripting
* webkit inspector
* themes
* search
* minimap
* multi-cursor
* completions
* ui: panel (search dialog)

# needs improvement
